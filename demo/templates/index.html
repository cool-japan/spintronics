About the Library

The spintronics library implements physical models from Prof. Eiji Saitoh's research (Univ. Tokyo/RIKEN) and the broader spintronics community. It provides pure Rust implementations of:

  • Spin Pumping - Generation of spin current from FMR
  • Inverse Spin Hall Effect (ISHE) - Spin-to-charge conversion
  • Spin Seebeck Effect (SSE) - Thermal spin current generation
  • Spin-Orbit Torque (SOT) - Current-driven magnetization switching
  • Topological Hall Effect - Berry phase from skyrmion textures
  • Rashba & Edelstein Effects - 2DEG spin-charge conversion
